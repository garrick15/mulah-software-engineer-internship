CSV Data Processor & Dynamic Table Renderer
A lightweight, vanilla JavaScript web application that parses uploaded CSV files, dynamically renders data into a stylized HTML table, and performs specific mathematical calculations based on row indices.

🚀 Features
1. File Handling Utilizes the FileReader API and PapaParse for CSV uploading
2. Dynamic UI Rendering
3. Automated Table
4. Responsive Styling

🛠️ Tech Stack
1. HTML5
2. CSS
3. JavaScript
4. PapaParse (Library): Used for robust CSV parsing to handle complex delimiters and edge cases.

📈 The Method
The application follows a four-stage pipeline to process the data:
1. Ingestion
2. Parsing
3. Transformation (Table 1)
4. Calculation (Table 2)
